# Privacy Policy

## Data Processing

This MCP server is a **read-only regulatory reference tool**. It:

- Processes no personal data
- Stores no user queries or session data
- Contains no analytics, telemetry, or tracking
- Makes no outbound network requests during normal operation (all data is served from a local SQLite database)
- Requires no user accounts or authentication

## Data Stored

The server's SQLite database contains only publicly available SFDA regulatory publications: regulatory framework texts, requirement descriptions, and guidance document content. No personal data is stored.
